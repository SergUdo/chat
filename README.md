## «Chat»

Use `rails 6.0.3.2` `ruby 2.6.5.`

## Launching

1. Clone or download this repo

2. Run in console:

```
bundle install
```

3. Create database:

```
rails db:create&&rails db:migrate
```

4. Start application:
 
```
 rails s
```

5. Visit to:

```
localhost:3000
```

Or open link: https://chatonrails.herokuapp.com/
